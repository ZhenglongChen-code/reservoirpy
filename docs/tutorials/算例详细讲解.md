# 油藏数值模拟器算例详细讲解

## 1. 系统初始化过程

### 1.1 创建网格
油藏数值模拟的第一步是创建结构化网格。网格是数值模拟的基础，它将油藏离散化为一系列控制体积（单元）。

```python
# 创建10×10×1的2D网格，单元尺寸为10m×10m×5m
mesh = StructuredMesh(nx=10, ny=10, nz=1, dx=10.0, dy=10.0, dz=5.0)
```

网格创建过程包括：
1. 生成网格节点坐标
2. 创建网格单元
3. 计算每个单元的几何属性（体积、中心坐标等）
4. 建立相邻单元关系

### 1.2 设置物理属性
物理属性定义了油藏中流体和岩石的特性。

```python
config = {
    'permeability': 100.0,  # 渗透率 100 mD
    'porosity': 0.2,        # 孔隙度 0.2
    'viscosity': 0.001,     # 粘度 0.001 Pa·s
    'compressibility': 1e-9 # 压缩系数 1e-9 1/Pa
}
physics = SinglePhaseProperties(mesh, config)
```

物理属性包括：
- **渗透率**：控制流体在多孔介质中的流动能力
- **孔隙度**：岩石中孔隙体积与总体积的比值
- **粘度**：流体的内摩擦系数
- **压缩系数**：流体和岩石对压力变化的响应

### 1.3 定义井模型
井是油藏开发中的重要组成部分，用于注入或采出流体。

```python
wells_config = [
    {'location': [0, 2, 2], 'control_type': 'rate', 'value': 0.001, 'rw': 0.05},
    {'location': [0, 8, 8], 'control_type': 'bhp', 'value': 1e6, 'rw': 0.05}
]
well_manager = WellManager(mesh, wells_config)
```

井模型支持两种控制方式：
- **定流量控制（rate）**：指定井的产量或注入量
- **定井底流压控制（bhp）**：指定井底压力

## 2. 离散化网格过程

### 2.1 控制方程
油藏数值模拟基于质量守恒定律和达西定律，其控制方程为：

∂(ρφ)/∂t + ∇·(ρv) = q

其中：
- ρ：流体密度
- φ：孔隙度
- v：达西速度
- q：源汇项

对于微可压缩流体，密度可表示为：
ρ = ρ₀ exp[c(P - P₀)]

### 2.2 有限体积法离散化
有限体积法将控制方程在每个控制体积上积分，得到离散化方程：

Vᵢ φᵢ cₜ (Pᵢⁿ⁺¹ - Pᵢⁿ)/Δt + Σ Tᵢⱼ (Pᵢⁿ⁺¹ - Pⱼⁿ⁺¹) = Qᵢ

其中：
- Vᵢ：第i个单元的体积
- φᵢ：第i个单元的孔隙度
- cₜ：综合压缩系数
- Pᵢⁿ⁺¹：第i个单元n+1时间步的压力
- Tᵢⱼ：第i个单元与第j个单元之间的传导率
- Qᵢ：第i个单元的源汇项

### 2.3 传导率计算
传导率表示相邻单元之间的流体传导能力：

Tᵢⱼ = (2KᵢKⱼ/(Kᵢ+Kⱼ)) × (A/d)

其中：
- Kᵢ, Kⱼ：相邻单元的渗透率
- A：界面面积
- d：单元中心距离

## 3. 组装系数矩阵

### 3.1 系数矩阵结构
离散化后的方程组可以写成矩阵形式：

A Pⁿ⁺¹ = b

其中A是系数矩阵，Pⁿ⁺¹是未知压力向量，b是右端向量。

### 3.2 矩阵元素计算
对于每个单元i：

**对角元素（i,i）：**
Aᵢᵢ = Vᵢ φᵢ cₜ/Δt + Σ Tᵢⱼ

**非对角元素（i,j）：**
Aᵢⱼ = -Tᵢⱼ

**右端向量元素：**
bᵢ = Vᵢ φᵢ cₜ Pᵢⁿ/Δt + Qᵢ

### 3.3 井项处理
井项作为源汇项添加到右端向量中：

- **定流量井**：bᵢ += q × Δt
- **定井底流压井**：bᵢ -= WI × (Pᵢⁿ⁺¹ - P_wf) × Δt

## 4. 求解过程

### 4.1 线性方程组求解
系统使用稀疏矩阵求解器求解线性方程组：

```python
# 使用双共轭梯度稳定法求解
new_pressure = discretizer.solve_linear_system(A, b, method='bicgstab')
```

支持的求解器包括：
- **bicgstab**：双共轭梯度稳定法（默认）
- **cg**：共轭梯度法
- **direct**：直接求解法

### 4.2 时间推进
模拟通过时间步进方式进行：

```python
while current_time < total_time:
    # 离散化方程
    A, b = discretizer.discretize_single_phase(dt, pressure, well_manager)
    
    # 求解压力场
    new_pressure = discretizer.solve_linear_system(A, b)
    
    # 更新压力
    simulator.update_cell_pressure(new_pressure)
    
    # 更新时间
    current_time += dt
```

## 5. 完整算例演示

### 5.1 问题描述
考虑一个20×20的二维油藏，中心有一口生产井，角落有一口注入井。

### 5.2 参数设置
```python
config = {
    'mesh': {
        'nx': 20, 'ny': 20, 'nz': 1,
        'dx': 10.0, 'dy': 10.0, 'dz': 5.0
    },
    'physics': {
        'permeability': 100.0,  # 100 mD
        'porosity': 0.2,
        'viscosity': 0.001,     # 1 cP
        'compressibility': 1e-9
    },
    'wells': [
        # 中心生产井（定井底流压）
        {'location': [0, 10, 10], 'control_type': 'bhp', 'value': 1e6, 'rw': 0.05},
        # 角落注入井（定流量）
        {'location': [0, 0, 0], 'control_type': 'rate', 'value': 0.001, 'rw': 0.05}
    ],
    'simulation': {
        'dt': 86400,           # 1天
        'total_time': 86400*365, # 1年
        'output_interval': 10,
        'initial_pressure': 30e6 # 30 MPa
    }
}
```

### 5.3 运行模拟
```python
# 创建模拟器
simulator = ReservoirSimulator(config_dict=config)

# 运行模拟
results = simulator.run_simulation()

# 获取结果
pressure_history = results['pressure_history']
time_history = results['time_history']
```

### 5.4 结果分析
```python
# 获取最终压力场
final_pressure = pressure_history[-1]

# 重塑为2D网格
pressure_2d = final_pressure.reshape(20, 20)

# 计算压力降
pressure_drop = 30e6 - final_pressure.min()

print(f"最大压力降: {pressure_drop/1e6:.2f} MPa")
```

## 7. 性能优化建议

### 7.1 网格优化
- 合理选择网格尺寸，平衡精度和计算效率
- 在井附近使用局部加密网格

### 7.2 时间步长控制
- 根据物理过程变化调整时间步长
- 使用自适应时间步长控制

### 7.3 求解器选择
- 对于小规模问题，使用直接求解器
- 对于大规模问题，使用迭代求解器
