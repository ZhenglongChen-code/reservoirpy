# 油藏数值模拟器项目使用指南

## 项目概述

油藏数值模拟器是一个轻量级、模块化的软件包，用于求解单相流和两相流渗流方程。该项目基于现有的`meshgrid.py`代码重构，在保持功能完整性的基础上，提供了清晰的模块化架构和扩展路径。

## 主要特性

- **模块化设计**：清晰的模块边界，易于维护和扩展
- **配置驱动**：YAML配置文件管理，支持不同模拟场景
- **功能完整**：支持2D/3D单相流和两相流模拟
- **用户友好**：简洁的API接口，丰富的可视化功能
- **工业标准**：符合实际工程应用需求

## 快速开始

```python

# 使用配置文件创建模拟器

# 运行模拟
results = simulator.run_simulation()
```

```python

# 使用字典配置
config = {
    'mesh': {
```

## 核心模块详解

### 1. 网格模块 (StructuredMesh)

网格模块负责生成和管理结构化矩形网格。

```python
from reservoir_sim import StructuredMesh

# 创建网格
mesh = StructuredMesh(nx=10, ny=10, nz=1, dx=10.0, dy=10.0, dz=5.0)

# 获取网格信息
print(f"网格尺寸: {mesh.grid_shape}")
print(f"总单元数: {mesh.total_cells}")
```

### 2. 物理属性模块 (SinglePhaseProperties)

物理属性模块封装了单相流动物理参数。

```python
from reservoir_sim import SinglePhaseProperties

# 创建物理属性
config = {
    'permeability': 100.0,  # mD
    'porosity': 0.2,
    'viscosity': 0.001,     # Pa·s
    'compressibility': 1e-9  # 1/Pa
}
physics = SinglePhaseProperties(mesh, config)
```

### 3. 井模型模块 (Well, WellManager)

井模型模块实现了Peaceman井模型，支持定流量和定井底流压控制。

```python
from reservoir_sim import WellManager

# 创建井
wells_config = [
    {'location': [0, 5, 5], 'control_type': 'rate', 'value': 0.001, 'rw': 0.05},
    {'location': [0, 15, 15], 'control_type': 'bhp', 'value': 1e6, 'rw': 0.05}
]
well_manager = WellManager(mesh, wells_config)
```

### 4. 离散化模块 (FVMDiscretizer)

离散化模块实现有限体积法离散化。

```python
from reservoir_sim.core.fvm_discretizer import FVMDiscretizer

# 创建离散化器
discretizer = FVMDiscretizer(mesh, physics)

# 离散化方程
A, b = discretizer.discretize_single_phase(dt, pressure, well_manager)
```

### 5. 主模拟器模块 (ReservoirSimulator)

主模拟器模块协调所有模块，提供用户友好的运行接口。

```python
from reservoir_sim import ReservoirSimulator

# 创建模拟器
simulator = ReservoirSimulator(config_dict=config)

# 运行模拟
results = simulator.run_simulation()
```

## 完整工作流程

油藏数值模拟的完整工作流程包括以下步骤：

### 1. 系统初始化
- 创建结构化网格
- 设置物理属性（渗透率、孔隙度、粘度、压缩系数）
- 定义井模型

### 2. 网格离散化
- 计算相邻单元之间的传导率
- 建立单元间的流动关系

### 3. 组装系数矩阵
- 构建线性方程组的系数矩阵A和右端向量b
- 处理边界条件和井项

### 4. 求解过程
- 使用稀疏矩阵求解器求解线性方程组
- 通过时间步进推进模拟过程

### 5. 结果分析
- 可视化压力场分布
- 分析井的生产动态
- 评估油藏开发效果

## 配置文件详解

配置文件使用YAML格式，主要包含以下部分：

### 网格配置 (mesh)
```yaml
mesh:
  nx: 20                    # X方向单元数
  ny: 20                    # Y方向单元数
  nz: 1                     # Z方向单元数
  dx: 10.0                  # X方向单元尺寸 (m)
  dy: 10.0                  # Y方向单元尺寸 (m)
  dz: 5.0                   # Z方向单元尺寸 (m)
```

### 物理属性配置 (physics)
```yaml
physics:
  permeability: 100         # 渗透率 (mD)
  porosity: 0.2             # 孔隙度
  viscosity: 0.001          # 粘度 (Pa·s)
  compressibility: 1e-9     # 压缩系数 (1/Pa)
  reference_pressure: 0.0   # 参考压力 (Pa)
```

### 井配置 (wells)
```yaml
wells:
  - location: [0, 5, 5]     # 井位置 [z, y, x]
    control_type: "rate"    # 控制类型: "rate" 或 "bhp"
    value: 0.001            # 控制值 (m³/s 或 Pa)
    rw: 0.05                # 井筒半径 (m)
```

### 模拟配置 (simulation)
```yaml
simulation:
  dt: 86400                 # 时间步长 (s)
  total_time: 31536000      # 总时间 (s)
  output_interval: 10       # 输出间隔
  initial_pressure: 30e6    # 初始压力 (Pa)
```
